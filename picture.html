Accessibility testing <picture> element and <srcset> attribute

Demo of <picture> and <source> elements, and <srcset> attribute

The two images below use a either a combination of the <picture> and <source> elements with a <srcset> attribute defined on the <source>, or the <srcset> attribute on the <img> element directly. Support for the <picture> element and <srcset> attribute is limited to Chrome Canary Chrome Canary 38+ and Firefox Nightly 34+ with the dom.image.picture.enabled and dom.image.srcset.enabled flags set to "true." Support for <srcset> also appears in Webkit 7.0.7+. Tests should still be performed on non-supporting browsers as should degrade gracefully, but do not expect alternative resources to be used (default image dimensions are 100px x 100px). In supporting browsers, changing the size of your viewport and reloading the page should result in the display of images resources with appropriate dimensions.
